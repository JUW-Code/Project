Course Description

User Interface (UI) and User Experience (UX) Design play key roles in the experience users have when interacting with digital products and appliactions. In this course, we'll cover the theory and methodologies behind UI and UX design. You'll also design your own wireframes and interactive prototype. Learning UI and UX basics can help you collaborate better on team projects and create new career opportunities.
